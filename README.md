### 📡 基于“一带多”增强模式的北斗差分路由器研究
*本项目旨在提升北斗差分定位的覆盖效率与可靠性。*

- **核心创新**：设计了“一带多”的增强通信模式，实现单个参考站为多个移动站提供高精度差分校正数据。
- **技术栈**：`北斗导航` `RTK` `差分定位` `嵌入式系统` `网络通信`
- **我的贡献**：负责了[例如：硬件平台搭建/通信协议设计/数据融合算法验证]等工作。
- **成果**：[例如：成功将定位精度从米级提升至厘米级，并显著降低了多用户场景下的系统复杂度与成本。]

### 🤖 LEGO EV3 智能小车全流程开发
*基于 LEGO EV3 平台，实现全自主导航的智能小车。*

- **核心功能**：完成了**颜色识别**、**黑线循迹**、**移动避障**与**定点旋转**等一系列任务。
- **技术栈**：`Python` `PID控制` `传感器融合` `超声波` `陀螺仪`
- **算法亮点**：运用**PID控制算法**，融合多传感器反馈，构建闭环控制系统，有效解决了传感器误判与电机滞后等工程难题。
- **成果**：实现了**零人工干预**的全程自主运行。
